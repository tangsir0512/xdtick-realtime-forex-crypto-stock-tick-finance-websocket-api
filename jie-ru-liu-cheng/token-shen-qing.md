# Token 申请

English / 中文

## 申请流程

### 第一步 - 填写注册信息

{% hint style="info" %}
Token申请地址：[https://xdtick.com/register](https://alltick.co/register)
{% endhint %}

{% stepper %}
{% step %}
### 访问注册页面

前往[Token申请地址](https://xdtick.com/register)以开始注册过程。
{% endstep %}

{% step %}
### 输入注册详情

* **邮箱地址**：输入一个有效的邮箱地址，该地址将用于后续的通讯和账户验证。
* **密码设置**：创建一个安全的密码。确保密码包含大小写字母、特殊字符和数字，以增加账户的安全性。
* **图片验证码**：输入图片上的验证码。如果没有看到验证码，请刷新页面。
{% endstep %}

{% step %}
### 完成注册

检查信息无误后，点击“注册”按钮。系统将处理您的注册信息，注册成功后，您将自动跳转到仪表盘页面。
{% endstep %}
{% endstepper %}

<figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

## Step 2 - 获取 API Token

{% stepper %}
{% step %}
### 进入仪表盘

注册完毕后，系统自动导航至仪表盘（Dashboard）。
{% endstep %}

{% step %}
### 找到Token

在仪表盘界面，导航至“API keys”部分。在这里，您将找到刚刚生成的API Token。
{% endstep %}

{% step %}
### 记录Token

复制并安全地保存您的Token。您需要在API调用时使用此Token进行身份验证。
{% endstep %}
{% endstepper %}

![仪表盘截图](<../.gitbook/assets/image (1).png>)

#### 注意事项

* 保护您的Token安全，避免共享给不信任的第三方。
* 如果遇到注册或Token生成过程中的任何问题，不要犹豫，立即联系客服支持获取帮助。
* 始终确保使用最新的Token进行API调用，以免因过期或失效而影响数据访问。

通过遵循上述步骤，您可以顺利完成Token的申请过程，从而获得访问API所需的权限。
